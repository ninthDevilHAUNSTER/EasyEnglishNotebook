# Easy English Notebook

## Introduction

希望能够扩展一些markdown的功能，方便我做一些论文笔记！

## English Words Extraction

提取方法

 > :notebook:  单词，分隔符为|中英文区分为;多释义用,隔开（notebook)

E.G.

```markdown
:notebook: tenacity:倾向|flat-out:全然地(adv)| fallacies:谬论
```

And run extract_word.py, U can get :

```csv
tenacity	倾向
flat-out	全然地(adv)
fallacies	谬论
```
